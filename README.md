# FBS축구센터 - 기본에 기술을 더하다

FBS축구센터의 공식 웹사이트입니다.

## 웹사이트 구성

- **메인 페이지**: 센터 소개 및 전체 정보
- **내 소개**: FBS축구센터 소개
- **훈련 내용**: 기본기, 전술, 체력 훈련 프로그램
- **센터 훈련 사진**: 실제 훈련 모습
- **축구양말 판매**: 센터에서 판매하는 축구용품

## GitHub Pages 배포

이 웹사이트는 GitHub Pages를 통해 호스팅됩니다.

### 배포 방법

1. 이 저장소를 GitHub에 푸시
2. GitHub 저장소 Settings > Pages에서 Source를 "Deploy from a branch"로 설정
3. Branch를 "main"으로 선택
4. 자동으로 웹사이트가 배포됩니다

## 파일 구조

```
football-soon/
├── index.html          # 메인 웹페이지
├── images/            # 이미지 파일들
│   ├── KakaoTalk_Photo_*.jpeg
└── README.md          # 프로젝트 설명
```
