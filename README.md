## 📸 이미지 교체 방법 (중요!)

현재 임시 이미지들이 들어있습니다. 실제 센터 사진으로 교체해주세요!

### 1️⃣ GitHub에서 이미지 교체하기

1. **GitHub 저장소 접속** → `images` 폴더 클릭
2. **교체할 이미지 클릭** → `Delete this file` 버튼
3. **새 이미지 업로드** → `Add file` → `Upload files`
4. **같은 파일명으로 업로드** (예: `KakaoTalk_Photo_2026-01-08-12-08-35 001.jpeg`)

### 2️⃣ 어떤 이미지를 넣어야 할까요?

| 위치 | 파일명 | 설명 | 권장 크기 |
|------|--------|------|-----------|
| **메인 배경** | `007.jpeg` | 센터 전체 모습, 대표 이미지 | 1920x1080 |
| **센터 소개** | `001.jpeg` | 센터 로고나 대표 사진 | 800x600 |

## ✏️ 내용 수정 방법

### 텍스트 변경하기
1. `index.html` 파일 열기
2. 수정하고 싶은 텍스트 찾기
3. 직접 수정 후 저장

### 주요 수정 포인트
```html
<!-- 센터 이름 변경 -->
<div class="logo">FBS축구센터</div>

<!-- 메인 슬로건 변경 -->
<h1>기본에 기술을 더하다</h1>

<!-- 센터 소개 내용 변경 -->
<p>FBS축구센터는 "기본에 기술을 더하다"라는 모토로...</p>

<!-- 제품명/설명 변경 -->

<p>고품질 소재로 제작된 전문 축구양말입니다.</p>
```

## 🚀 배포 방법

### GitHub Pages 설정
1. **GitHub 저장소** → **Settings** 탭
2. **Pages** 메뉴 클릭
3. **Source**: "Deploy from a branch" 선택
4. **Branch**: "main" 선택
5. **Save** 클릭

변경사항을 GitHub에 푸시하면 자동으로 웹사이트가 업데이트됩니다!
